单腿标定：θ / α / γ 量哪个角、怎么量

三个关节角量的都是虚拟连线对参考系（另一连线 / 重力 / 机身）的夹角，不是零件实体边。θ 已完成（86.4° 已入 config）；剩 femur 的 α 和 coxa 的 γ（§04、§05）。

F

femur 舵盘中心螺丝（髋俯仰轴）。femur 舵机输出轴上那颗中心螺丝的圆心。

K

tibia 舵盘中心螺丝（膝轴）。§3.4 已确认舵盘中心 = 膝轴。

P

吸盘唇口圆心（自由态、不通真空）。KP 的实测值同时就是这个口径下的 tibia_len。

01θ 量法：三边定角，不碰量角器 ✅ 已完成

  1. 跑 software/scripts/servo_calib_helper.py，敲 17 1510（L1 tibia 到标定中心；脚本启动默认打 1500，必须手动改到 1510）。
  2. 通道 15/16（coxa、femur）摆到方便下卡尺的位置即可——F、K 都在 femur 刚体上，θ 只由膝舵机决定。全程保持使能，不许手掰。
  3. 量三条点到点直线距离：① FK（应 ≈80mm，当自检）② KP ③ FP。螺丝中心对中心，多测两次取平均。
θ = arccos( (FK² + KP² − FP²) / (2 · FK · KP) )

误差参考：边长差 1mm ≈ 角度差 1°。判 44° 基准问题绰绰有余；要填 config（±1~2°）就用卡尺仔细量。

02不要这样量

  • 量角器贴 A 段或 B 段的任何实体边——θ 是虚拟线 K→P 相对 K→F 的夹角，零件上很可能不存在平行于 K→P 的面（§4.2）。
  • 把「A 段与 femur 成一条直线」当作 θ=180° 的参考位——那个姿态的 θ 实际是 180°−δ′（δ′ = A 段方向与 K→P 的固定夹角）。
  • 拿方轴 / 方盒側面当基准面——方轴方向与 K→P 差约 20°（图中 δ，§4.1）。

04femur：α_center 高差法 ✅ 结果：49.7°

2026-07-17 实测 Δh = h_K − h_F = 61mm → α_center = arcsin(61/80) = 49.7°， 正落在官方 35° + 一个花键齿 14.4° = 49.4° 上（差 0.3°），已入 config。摆位图与方法留档。

先对齐术语（图中标注）：

  • L1 = 整条左前腿（coxa + femur + tibia-suction 一体件），L=左、1=前，通道 15/16/17。不是一个点。
  • 髋 = 腿根的两个轴：C（coxa 轴，竖直，管水平摆）+ F（femur 轴，水平，管抬升）。「膝在髋上方 / 吸盘在髋下方」里的髋指 F。
  • 膝盖 = K（tibia 舵盘中心螺丝）；吸盘圆心 = P。与 θ 三边法用的是同样的点。
α_center = arcsin( (h_K − h_F) / 80 )　　（K 高为正；80 = 实测 FK）
  1. 机身放平（手机水平仪 ≤0.3°），L1 悬出桌沿但让 F、K 都还在桌面上方（如图，只有吸盘出檐）。敲 16 1510，全程使能。
  2. 量桌面 → F 螺丝顶、桌面 → K 螺丝顶。同型号螺丝，头半径在差值里抵消。精度：±0.5mm → ±0.6°。
  3. 交叉验证：femur 有验证过与 F→K 精确平行的长直边（§2.5），水平仪先在机身归零再贴边读数，应与高差法一致（1° 内）。
  4. 填法：femur attach_deg := α_center，robot.py 直接传 α，无基准换算。期望值 35 ± n·14.4°。

05coxa：γ_center 俯视投点法 ⏸ 已决定推迟到 P3

2026-07-17 决定：推迟到 P3 装机身时测。P0 无机身，γ 的 0 点（中性方向 55°）物理上不存在； P1/P2 单腿台架不依赖 γ 绝对值（压力闭环，偏差只是横移 ~2.3mm/°）；P3 届时直接用文末「复核版」基线法（不依赖口袋对准假设，更硬）。 下面的舵机壳基准法留档备用——如果 P2 期间发现横向瞄不准，可以随时用它先量个临时值。

装上机身后的复核版（留档）——用 L1↔R1 舵盘螺丝投点连线当基准，不依赖"口袋对准 55°"的假设。现在没有机身，暂时用不了：

  1. coxa 舵机壳底面平贴桌面并夹紧或压重（腿是悬臂，会拉歪没固定的壳），壳下压方格纸。敲 15 1510。
  2. 基准线：沿壳的长边贴纸描一条直线——平行于壳轴线，就是 γ 的 0° 参考；正方向取指向输出轴那端。
  3. 腿方向线：coxa 不动，femur/tibia 摆两个不同伸展度，各用直角尺把吸盘唇口圆心竖直投到纸上标点，两点连线（取点纪律同 θ/femur）。
  4. 纸上取远点读坐标算 atan2 求两线夹角（别用小量角器）：γ_center = 腿线相对壳轴线的角，俯视逆时针为正。期望值 −8 ± n·14.4°。填 coxa attach_deg := γ_center。
  5. 前提与复核：此法假设机身口袋把壳轴线对准中性方向 55°（官方设计意图）。装上机身后用上面『复核版』的基线法复核；不急的话也可以整体推迟到装机身时再测——单腿台架（P1/P2）用不到 γ。三个 attach 齐后做 FK 抽查（发关节角、量足端 vs leg_fk，<5mm 通过）。
