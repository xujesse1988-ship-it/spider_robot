## 第 0 步 · 一个必须先想清楚的构型问题（半小时，影响打印清单）

ROADMAP 的分阶段设定是：**P3 地面行走用标准小腿**（left/right-tibia + 足底开关 +
橡胶足尖），**P4 才换吸盘小腿上墙**（"6 只吸盘足上机"）。但这带来两个代价：

1. **L1 现在装的是吸盘小腿**（P2 用的、已标定）。按 ROADMAP 走，P3 要把它换成
   标准小腿——换结构件 = tibia 标定作废，P4 换回来还要再标一次。
2. 全部 6 条腿的 tibia 标定在 P4 换吸盘小腿时**全部作废重做**。

两条路线：

| 路线 | 内容 | 代价 | 收益 |
|---|---|---|---|
| **A · 按 ROADMAP** | 6 条腿全用标准小腿 + micro-switch-tip + 橡胶足尖 | 多打 6 个标准小腿（小件，~13g/个）；tibia 标定 P3/P4 各做一遍 | 足底开关触地检测（官方成熟路线）；吸盘唇口不磨地 |
| **B · 直接吸盘小腿** | 6 条腿直接装吸盘小腿走地面 | 无触地开关（吸盘小腿末端是吸盘腔，没有开关位）；唇口当脚掌磨地（吸盘是 ¥5~15 耗材，备了 8 个） | tibia 只标一次；L1 标定不作废；P4 不用再换件 |
| **B 需要正视的问题** | 足底开关 BOM 里标注"爬墙步态刚需"，但吸盘小腿本来就没有开关位——**P4 无论如何要靠压力传感器做触地/吸附确认**。开关其实只服务 P3 | | |

**✅ 已定路线 B（2026-07-22）**：6 条腿直接装吸盘小腿走地面。P3 验收里的
"触地检测步态"降级为开环三角步态 + 压力/电流兜底（已记入验收核对表）；
足底开关仍随第二批购入，留 P4 实验。

## 第 3 步 · 装配（快递到货后，3~5 天）

1. 顺序照官方 YouTube 装配视频（频道 MakeYourPet），接线照
   `hardware/makeyourpet-hexapod/wiring-diagram-servo2040.png`，通道映射在 `config.py`。
2. **装舵盘规矩**：先 `python scripts/servo_center.py` 全舵机回中再装舵盘。
   注意：servo_center 发 1500µs，而标定基准点是 1510µs（`(1040+1980)/2`，
   §4.6）——装舵盘用 1500 没问题（花键一齿 ~14°，远粗于 1°），但**后续量
   attach_deg 时必须用 `servo_calib_helper.py` 发 1510**。
3. **舵机使能状态下永远不许手掰**——35kg·cm 堵转打齿，且掰出来的角度是假的。
4. 供电老规矩：Pi 独立 5V/5A，绝不与舵机共轨；舵机 7.4V 经 Servo2040 A0/GPIO26
   继电器；装完 `vcgencmd get_throttled` 必须 0x0。
5. 走线时给 P4 留量：每条腿根部预留真空软管的走线通道和扎带位（现在不接管，
   但线束别把路堵死）。

## 第 4 步 · 全舵机标定（装配后必做，2~3 天，最费事的工序）

顺序有讲究（`LEG-GEOMETRY-OPEN.md` §5 剩余主线）：

1. **coxa γ（装机身后第一件事）**：用基线法——L1↔R1 舵盘螺丝投点连线为基准，
   不依赖"机身口袋对准 55°"（§5 主线 1，方法图见标定参考图 §05）。
   L1 的 coxa 一直用官方 -8 凑合，现在终于能实测了。
2. 每关节 ±45° 脉宽：手动（发脉宽,不是手掰）摆到 -45°/+45°，记录填
   `config.py` 的 `us_m45/us_p45`；方向反的 `sign=-1`。
3. femur/tibia `attach_deg` 逐腿实测：照 L1 的成熟方法（femur 高差法 §2.12、
   tibia 三边法 §2.11）。**未实测的腿沿用官方 68 有 §2.9 的 44° 基准风险，
   六条腿全要实测，别偷懒。**
4. **端到端 FK 抽查（P3 验收项）**：每腿发一组关节角，卷尺量足端实际位置 vs
   `leg_fk` 预测，**<5mm 通过**。L1 顺带裁决 §4.4 记录的 tibia_len 116 vs 120
   之争（4.6mm 张力正好在判据边上，量三个不同姿态取多数）。
5. 规矩重申：换结构件/重装舵盘 → 该关节标定作废重做。

## 常见问题

| 症状 | 处理 |
|---|---|
| 站立时某腿姿态明显不对 | 先查 sign 再查 attach_deg;某关节差 ~14° 的倍数 = 舵盘装错齿 |
| 行走跑偏 | 左右腿 attach_deg 不对称;FK 抽查数据里找偏差最大的腿 |
| 走几步后舵机烫 | 站姿关节角别贴行程边缘;步态放慢(SPEED 调低);检查是否某腿承重远大于其他 |
| Pi 重启 | 欠压。`get_throttled` 非 0 → 供电问题,先修再继续,这在 P4 是掉落事故 |
| 舵机偶发抖振 | 信号线与舵机电源线捆太近;分开走线或加磁环 |
| 步态相位乱 | 通道映射错位,`config.py` 对照接线逐路核 |

## 安全

- 落地测试前全部架空跑通;初次落地在软垫上。
- 舵机使能中不许手掰任何关节;调整姿势先断使能。
- 锂电规矩同 P0~P2:充电不离人、XT60 随手可拔、先断电再改线。
- 摔机快修:备件腿常备(第 2 步最后一批),摔坏当天能换。
