# P2 单腿爬墙试验 · 详细操作指南

目标（第 5~9 周）：**整个项目的决策门——一条腿在竖直玻璃前自动完成
"伸腿 → 压紧 → 抽气 → 压力确认 → 承力 → 放气 → 抬腿"循环，并独立撑住 1.5kg 配重。**
本阶段结束时你应该能拍出两段视频：①连跑 50 次吸放循环的台架；②滑车锁销拔掉后，
1.5kg 配重完全挂在吸附的腿上 5 分钟纹丝不动。

通过之前**不买第二批器件**。顺序建议：第 1 步（备料）先行；第 4 步（软件干跑）不依赖
台架，等快递期间就能做完；第 2~3 步搭台，第 5~6 步是正戏，严格按序。

前置状态核对（都是 P0/P1 已完成项，缺一个先回去补）：

- [ ] L1 腿 ±45° 脉宽 + `attach_deg` 实测值已入 `config.py`（tibia 93.6 / femur 49.7，
      见 `LEG-GEOMETRY-OPEN.md` §2.11/§2.12；coxa 用官方 -8 即可，本台架不敏感，见第 2 步）
- [ ] `Pi5VacuumIO` 实装且 P1 三条曲线达标（<1s 到 -40kPa；60s 泄漏不过 -20；释放 <0.5s）
- [ ] 吸盘已装进 `left-tibia-suction.stl` 腔体、门盖已粘，整足挂重复测通过（P1 第 9 步）

## 第 2 步 · 台架几何（半天~1 天）

```
        立柱(竖直)                玻璃(竖直,背衬板)
          │ ┌滑轨                    ║
          │ │┌滑车板──coxa舵机壳═══╗  ║   coxa 输出轴水平、垂直指向玻璃
          │ ││   +配重      femur ╲ ║
          │ ││锁销↕                ╲║ ← 吸盘压在玻璃上
          │ ││                 tibia║
          ┴─┴┴──缓冲垫──────────────║──
```

1. **玻璃竖直**、擦净（酒精），背面衬板夹紧。
2. **coxa 舵机壳固定在滑车板上**：输出轴水平、垂直指向玻璃；
   壳安装面（相当于整机的机身平面）**距玻璃 90mm**。座上开长槽做 ±20mm 可调。
   > 为什么是 90 而不是 ROADMAP 里写的"约 130"：130 是标定前拿"足端工作半径"凑的
   > 粗值；按实测标定 + 物理吸盘轴（δ=22.7°，见第 4 步）重算（`CLIMBING-DESIGN.md`
   > §6 A 表），站高 90mm、支撑相 reach 170mm 时吸盘轴偏角仅 1.6°。
   > 本指南的 90mm 为准，脚本第 4 步的常量按它算好了。
3. **腿平面竖直**：coxa 回中（1510µs）时，femur/tibia 的摆动平面应铅垂
   （目测+手机水平仪贴 femur 侧面即可，±3° 无所谓）。
4. **coxa 没标定不影响本台架**：coxa 轴⊥玻璃，γ 偏差只是让足端沿玻璃面转过一段弧，
   吸盘轴相对玻璃法线的姿态完全不变（这正是把 γ 实测推迟到 P3 的依据，
   `LEG-GEOMETRY-OPEN.md` §5）。腿偏着伸出去别扭的话，微调 `config.py`
   coxa `attach_deg` 把它扶正，数值不用精确。
5. **刚性验收**：手推腿根，任何方向晃动 <1mm。压紧反力是几十牛级别，
   支架软了预压行程会被变形吃掉，吸不住的第一嫌疑人就是它。
6. **几何自检**（先别装玻璃）：跑第 4 步的 `--mock` 确认软件后，真机发 press 姿态，
   卡尺量吸盘唇口平面到"玻璃将要安装的平面"——唇口应**越过**该平面约 15mm
   （这就是压紧过盈）。差得多就动长槽调 90mm 那个距离。

## 第 3 步 · 滑车承力机构（半天）

1. 滑轨竖直装立柱上，滑车行程 ≥50mm，下端放缓冲垫。
2. 腿根座+配重全部装滑车板上，**总滑动质量称到 1.5kg**（≈ 整机 2.5kg ÷ 3 只
   有效承力足，加余量）。配重尽量压在滑轨轴线上，偏了会卡轨。
3. 锁销两个状态：**锁死**（滑车固定 = 刚性台架，跑循环用）/
   **解锁**（滑车自由下滑 = 配重全部转移到吸附的腿上，承力验收用）。
4. 演练一次"单手拔销、单手上销"，拔销时另一只手不碰滑车。
5. 脱吸演练（无腿空滑车）：解锁让滑车自由落到缓冲垫，确认行程末端不撞坏东西。

## 第 4 步 · 软件与干跑（等台架期间做完）

软件交付物已就位：`software/scripts/single_leg_wall.py`。

**几何常量**（脚本开头，按第 2 步的 90mm 台架算好）：

| 常量 | 值 | 含义 |
|---|---|---|
| `FOOT_X` | 169 | 足端径向位置：压紧时**物理吸盘轴**⊥玻璃（偏 0.1°） |
| `Z_LIFT` | -70 | 抬离位，唇口离玻璃 20mm |
| `Z_CONTACT` | -90 | 唇口刚触玻璃（= 髋平面距玻璃距离） |
| `Z_PRESS` | -105 | 压紧位 = 触壁再进 15mm（预压行程 13.5 + 预载 1.5，`LEG-GEOMETRY-OPEN.md` §4.4 实测） |

> ⚠ **FOOT_X 为什么是 169 而不是"K→P 垂直玻璃"的 122**：物理吸盘轴沿方轴方向，
> 与模型虚拟线 K→P 差固定 **δ=22.7°**（膝轴 K 不在方轴上，`LEG-GEOMETRY-OPEN.md`
> §2.13）。按 K→P⊥玻璃（122）去压，吸盘会以 22.5° 斜角怼玻璃，超 15° 容差直接吸不住。
> 台架图：见 `html/p2-rig-diagram.html`。

台架距离不是 90 时改常量速查（H = 壳安装面到玻璃实测值）：
`Z_CONTACT=-H`、`Z_PRESS=-H-15`、`Z_LIFT=-H+20`；`FOOT_X`：H=80→168、90→169、100→169。
整个循环内物理吸盘轴偏离玻璃法线 ≤4.5°，在 P1 实测的 15° 容差内，行程余量 ≥44°。

1. **mock 干跑**（开发机或 Pi 都行）：
   ```bash
   cd ~/spider/software && source .venv/bin/activate
   python scripts/single_leg_wall.py --mock --cycles 3
   # 期望：3 个循环全部成功，保持期压力 -40kPa 左右，成功率 100%
   ```
2. **参数**：`--cycles N` 循环数；`--hold 秒` 每循环吸附保持时长；`--csv 路径` 明细
   （20Hz 的 t/cycle/phase/kpa/amp，压力曲线和舵机电流都在里面）；`--release` 善后。
3. **失败重试**已内建：SUCKING 超时 → 抬 5mm → 重新压紧，最多 3 次，全败记该循环失败。
4. **Ctrl-C 是"冻结"不是"停止"**：阀保持当前状态、舵机保持使能——吸附中的滑车
   不能因为你按了 Ctrl-C 就掉下来。善后固定跑：
   ```bash
   python scripts/single_leg_wall.py --release    # 放气 + 回抬离位
   ```
5. **泵策略**：脚本屏蔽了状态机的泵滞环（P1 单传感器下罐压读数失准会让泵长转发热），
   改为"抽气前预抽 1.5s + 吸附保持期低于 -45kPa 补抽"。泵占空比因此很低，
   50 循环连跑不担心过热。
6. 真机首跑前把 P0 的老规矩过一遍：跳线已割、电池极性、`get_throttled` = 0x0。

## 第 5 步 · 分级上量（1~2 天）

每级通过再进下一级，任何一级不过先解决再前进：

| 级 | 状态 | 命令/操作 | 通过标准 |
|---|---|---|---|
| 1 | 锁死 + 无配重 | `--cycles 1 --hold 3` | 一次成功；屏幕吸附时间 ≈ P1 的 <1s |
| 2 | 同上 | `--cycles 10` | 成功率 10/10（允许内建重试） |
| 3 | 锁死 + 配重上滑车 | `--cycles 5` | 结构无变形、成功率不降 |
| 4 | **解锁承力** | `--hold 300` 单循环，屏幕确认吸附后手动拔销 | 1.5kg 悬挂 5 分钟不脱落，压力稳定 |
| 5 | 侧向载荷 | 承力状态下弹簧秤水平拉滑车至 1kg，保持 10s | 不脱落、不滑移 |
| 6 | 力矩余量 | 看 CSV：压紧瞬间电流峰值 vs 保持期基线 | 峰值有肉眼余量；把 `Z_PRESS` 再加深 3mm 复测电流增幅做参照 |

- 第 4 级操作纪律：**屏幕没显示吸附确认（≤-30kPa）绝不拔销**；拔销后人站侧面；
  重新上销后才允许放气。
- 第 6 级说明：Servo2040 读的是全板总电流，是近似值——看的是相对变化不是绝对值。
  35kg·cm 舵机按 50% 降额的判断标准：加深 3mm 预压后电流增幅仍平缓（没有陡增/异响/
  发烫），即认为墙面姿态有余量；陡增说明已接近堵转区。

## 常见问题

| 症状 | 处理 |
|---|---|
| SUCKING 全超时，重试 3 次全败 | 罐没抽起来（听泵声、查 12V）；预压不足——量第 2 步几何自检那 15mm 过盈；玻璃或唇口脏 |
| 压紧时舵机嗡嗡响、电流大 | H 距离量错导致 press 过深；支架变形吃预压（回第 2 步刚性验收）；`Z_PRESS` 临时改浅 3mm 定位问题 |
| 吸附成功但拔销就掉 | 真空度贴着 -30 线过的——查泄漏（P1 曲线 2 复测）；把脚本判据从 -30 收紧到 -40 再验 |
| 滑车解锁后不下滑（假承力） | 滑轨卡滞/配重偏置卡轨——空滑车复演第 3 步第 5 条 |
| 承力时缓慢滑移 | 剪切贴边缘了：玻璃重新酒精擦；真空度提到 -50 以下；仍滑就是 30mm 吸盘剪切极限，考虑 40mm |
| 循环成功率在 90% 附近晃 | 逐条看 CSV 失败循环的压力曲线：抽不下去=密封问题（几何/清洁），抽下去又回升=泄漏 |
| 电流列恒定不变 | mock 下固定 1.0 属正常；真机检查是不是没接舵机电源 |
| Ctrl-C 后想继续 | 先 `--release` 善后回到已知状态，再重新起 `--cycles` |
| 吸附中断电了会怎样 | 取决于 P1 第 3 步实测的阀断电行为；"断电≠保持"的阀会直接放气掉落——测试中途不要动电源 |

## 安全

- **滑车和配重下方任何时候不放手脚**，缓冲垫全程在位；解锁承力时人站台架侧面。
- 吸附确认（屏幕 ≤-30kPa）之前绝不拔锁销；放气之前必须先上销。
- 玻璃背面衬板、边缘贴胶带；挂重和侧拉时戴护目镜（吸盘崩脱会弹）。
- Ctrl-C 不放气是刻意设计——善后只用 `--release`，不要直接拔舵机电。
- 12V/锂电规矩同 P0/P1：先断电再改线、充电不离人、XT60 快断随手可拔。
