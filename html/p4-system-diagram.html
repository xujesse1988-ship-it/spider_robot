阀的状态语义（本项目要求的接法）
阀状态气路接通含义
断电（默认）吸盘 ↔ 真空母线抽气 / 补抽 / 保持——断电不掉落
通电吸盘 → 大气（母线侧被切断）释放该足，准备迈步
⚠ 装配前必测：市售 0520B 的端口语义常与此相反（断电 = 通排气）。 装配时必须逐阀在台面实测断电行为（断电时哪两个口通）；若相反，把真空接到阀的"排气口"反用。 断电 = 放气的接法上墙，等于断电即坠落。
关键规则
  • Pi 独立供电：舵机堵转的压降绝不能重启 Pi——爬墙中重启 = 掉落。装好后 vcgencmd get_throttled 确认无欠压。
  • 接点 = 泄漏点：全链路统一 4mm 宝塔 + 扎带锁紧，密封用环氧、不用热熔。
  • P4 禁用面包板 / 杜邦插接——震动 = 间歇断连 = 状态机误判；分压板洞洞板焊接 + JST / 焊线。
  • 单向阀装在泵与储气罐之间，方向只许"罐→泵"——防泵停机后经泵膜倒漏。
  • 储气罐 = 真空电池：新脚秒吸、削峰、泵故障缓冲——网络容积比纯管路大 50 倍，泄漏衰减慢 50 倍。
  • 步态铁律：新脚压力确认 ≤ −30kPa 之前，不放开下一只脚（始终五足支撑）。
