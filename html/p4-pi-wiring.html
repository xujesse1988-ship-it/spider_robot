规则
  • 单点共地：12V / 舵机功率地与 Pi 信号地只在 MOSFET 板汇接一次，别处不再相连——成环 = 地环流搅乱 ADS 读数。
  • 上电顺序：先只给 Pi 上电 → vcgencmd get_throttled 必须 0x0 → 再合舵机继电器与 12V 轨；改线后重查。
  • P4 禁面包板 / 杜邦：header 出线用带壳压接排线或焊接 + 热缩；MOSFET 控制线与 12V 功率线分开走束。
  • 阀控电平：adhesion.py 实测 VALVE_ON_LEVEL=0（低电平 = 吸盘接真空）——P4 六路阀逐路实测后再写映射。
