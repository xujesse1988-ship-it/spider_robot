# 爬墙方案设计论证

## 1. 吸附方式选型

| 方式 | 原理 | 适用墙面 | DIY 可行性 | 结论 |
|---|---|---|---|---|
| **真空吸盘 + 阀组** | 泵抽真空，每足独立阀控吸放 | 玻璃、瓷砖、烤漆、亚克力 | ★★★★☆ 全部标准气动件 | **主线** |
| EDF 负压压墙 | 涵道风扇把机器压向墙面 | 几乎所有平整墙面 | ★★★★☆ 航模件即得 | 放弃：只对 <800g 平台成立，与 2kg 大平台不兼容（见 §3） |
| 微刺（microspine） | 细针钩挂粗糙表面微观凸起 | 砖、水泥、粗糙涂料 | ★★★☆☆ 针+打印件即可，但单靠它承重需精细的柔性悬挂 | 放弃：需与 EDF 搭配才实用 |
| 磁吸 | 电磁/永磁 | 仅钢铁表面 | ★★★★★ | 场景太窄，放弃 |
| 静电吸附 | 高压电极极化墙面 | 多种 | ★☆☆☆☆ 高压+吸附力弱 | 研究级，放弃 |
| 仿生干胶（壁虎） | 范德华力 | 光滑面 | ★☆☆☆☆ 材料自制困难、易脏 | 研究级，放弃 |
| 热熔胶足 | 熔胶粘-加热脱 | 多种 | ★★☆☆☆ 循环慢、能耗高 | 放弃 |

### 腿数选型：为什么是六足而不是四足

四足省 6 舵机 + 2 套足模块（约 -450g / -¥300~500），但在墙上严格更危险：
摆动相吸附足数六足=5、四足=3；单吸盘脱吸（P2 验收即允许 5% 失败率，必然发生）后
六足剩 4 足可优雅降级重吸，四足剩 2 足承受动态冲击载荷，大概率连锁脱吸坠落。
另外四足单足峰值载荷 ×1.5（吸盘需升 40mm，重量优势缩水）、地面静步态要摆重心、
地-墙过渡只有"2+2"支撑。学术先例（広瀬 NINJA-I 四足吸盘爬墙）依赖研究级的
专用阀与高自由度腿，不适合 DIY。
**决定：六足。可逆后手：官方配置有 MODE_QUADRUPED，步态引擎加四足步态即可用
同一硬件 A/B 实测；若 P4 减重卡死，拆两条中腿转四足是随时可做的实验。**

## 2. 吸盘足方案力学预算

以 MakeYourPet 大平台 + 吸附系统整机 **m = 2.5kg**（余量上限）计：

- 单只 30mm 吸盘（配合 3D 打印抗形变外壳），工作真空度 -50kPa 实测：
  法向拉脱力 = **25N（2.5kg）**，竖直玻璃剪切拉脱力 = **15N（1.5kg）**（约为法向的 60%）。
- 爬墙步态下任意时刻最少 **5 只吸附**，极端保守按 3 只发生有效承力计算：
  法向总拉力 ≥ 75N，对 2.5kg(25N) 重力的**法向安全系数 = 3.0**
  剪切总拉力 ≥ 45N，对 2.5kg(25N) 重力的**剪切安全系数 = 1.8**（若按 5 只有效，则剪切安全系数达 3.0，余量非常充足！）
- 剥离力矩：重心离墙 d，上排吸盘受额外拉力 ≈ m·g·d/h（h 为上下足距）。
  d 每减 1cm 都是实打实的余量 → **爬墙姿态身体尽量贴墙、电池等重物靠墙侧安装**。

系统结论：
1. 30mm 吸盘 ×6 + -50kPa 对 2.5kg 内的整机成立；负重阶段换 40mm（面积 ×1.78）。
2. 泵只需维持储气罐真空，阀负责各足吸放 → 泵可间歇工作省电。
3. **常闭阀 + 单向阀**让断电状态=全足吸附锁定，这是最重要的被动安全设计。

## 3. 已放弃路线的论证记录：EDF 压墙

粗糙墙面（砖、水泥、乳胶漆）的常见 DIY 方案是涵道风扇把机器压向墙面 + 摩擦/微刺足。
放弃原因（平台定为 MakeYourPet 大平台后由物理否决）：70mm 12 叶 EDF @4S 静推力仅
12~18N；摩擦系数 μ≈0.6 时防滑需 F_press ≥ m·g/μ，2kg 整机需要 33N+ 的持续压紧力，
意味着双 90mm EDF、数百瓦功耗、且风扇一停就掉——与"静态停留、可负重"的项目目标冲突。
此路线仅对 <800g 的轻量平台成立（参考文献见 §7 的 GECO 与 IJERT 论文）。
**结论：本项目只做光滑墙面（玻璃/瓷砖/烤漆），粗糙墙面不在范围内。**

## 4. 气路图

```
泵(555×2 并联) → 单向阀 → 储气罐(PET瓶) ─┬→ 常闭三通阀#1 → 足1吸盘
      ↑                    │(XGZP6847A     ├→ 常闭三通阀#2 → 足2吸盘
  MOSFET板←Pi5 GPIO        │ 压力反馈,      ├→ ...
           Pi5 I2C←ADS1115←┘ 经ADS1115)    └→ 常闭三通阀#6 → 足6吸盘
   三通阀第三口 = 大气（通电切换到放气位，释放该足）
```

控制逻辑（吸附状态机，每足独立）：
`RELEASED → 压紧墙面 → 阀切吸气位 → 等待压力 ≤-30kPa（超时500ms则抬脚重试）→ ATTACHED（允许承重）→ 需要移动时阀切放气位 → 延时300ms → RELEASED`

步态约束：**任意时刻至少 5 足处于 ATTACHED**，且相邻足不同时释放。

> [!TIP]
> **电磁阀断电保压的神级接法 (P1 阶段总结)**
> 采购的 0520B 类常闭三通阀实测行为：断电时孔 1 与孔 3 连通，孔 2 封闭。
> 为了实现“断电 = 吸盘保持真空”的被动安全设计，**必须采用以下接法**：
> 1. 孔 1 ➡️ 接吸盘
> 2. 孔 3 ➡️ 接真空泵抽气口
> 3. 孔 2 ➡️ 放空通大气
> **原理解析**：断电时（1通3），吸盘连着停转的真空泵，利用隔膜泵内部的单向阀实现死锁保压。通电时（1通2），吸盘接通大气孔泄压抬腿。此接法不仅实现了绝对的断电防坠落安全，而且在静止吸附时电磁阀 0 耗电。代码中需配置 `VALVE_ON_LEVEL=0`。

## 5. 电气架构演进

**单脑架构：树莓派 Pi 5 从第一天就是大脑**（`software/` 自研软件，全阶段同一套代码栈）：

```
2S 电池 ─┬─ 7.4V 直供 → Servo2040 舵机电源轨(经继电器) → 18 舵机
         ├─ 5V/5A 降压 → Pi 5 (USB-C，独立供电，绝不与舵机共轨)
         └─ XL6009 升压 12V → MOSFET板 → 6 阀 + 2 泵
Pi 5 ─USB──→ Servo2040（chica 串口协议：18 舵机 + 6 足底开关 + 电压/电流）
Pi 5 ─I2C──→ ADS1115 ×2 → XGZP6847A 压力传感器（P1 起）
Pi 5 ─GPIO─→ MOSFET 板（阀/泵开关，P1 起）
Pi 5 ─CSI──→ 摄像头模块（P5，FPV）
```

- 协议说明与驱动实现见 `software/hexapod/driver.py`（已按固件源码逐字节核实）。
- 步态每挪一步前先由吸附状态机确认"目标足已吸附"（`software/hexapod/adhesion.py`）。
- **必装低压报警**：锂电欠压=泵停=吸附衰减=掉落。软件层 `check_power()` 在 6.0V 以下直接抛异常停机（阀常闭，断电仍保持吸附）。

## 6. 活动度分析：3DOF 腿能否上垂直墙

分析脚本 `tools/workspace_analysis.py`（用真机连杆 43/80/120mm——tibia 为 L1 实测值，
见 LEG-GEOMETRY-OPEN §2.11——与舵机电气行程 femur -40.3°~+139.7°（L1 实测 attach 49.7°，
§2.12）、膝内角 0°~176.4° 计算；2D 腿平面模型，保守。2026-07-17 按实测标定重算）。
3DOF 腿的本质限制：**足端位置与吸盘朝向耦合**——吸盘轴向 = tibia 末段方向
a_t = α + θ - 180°，选定落足点后 a_t 就定了，只能靠波纹吸盘的角度顺应性兜底。

**A. 在墙上行走：可行。** 身体平行墙面时与地面行走运动学等价，唯一新约束是
吸盘轴线相对墙面法线的偏角（波纹吸盘容差约 15°）：

| 站高（身体离墙） | 步幅 | 支撑相内偏角 |
|---|---|---|
| 90mm | 0 / 30 / 40mm | 6.0° / ≤12.9° / ≤15.1° |
| 70mm | 0 / 30mm | 11.0° / ≤17.3° |

→ 爬墙步态用 **站高 90mm + 步幅 ≤30~40mm**，偏角在容差内。注意"身体贴墙"
（降剥离力矩）与"吸盘对正"（90mm 更好）方向相反，90mm 是平衡点。

**B. 地-墙过渡（前足上墙）：运动学上有解。** 前腿可落足区（偏角容差 15°，
相对前髋的高度范围，mm）：

| 身体俯仰 | 墙距前髋 100 | 140 | 180 |
|---|---|---|---|
| 0°（平身） | +25~+85 | +47~+107 | +47~+107 |
| 30°（绕后髋抬头） | +52~+112 | +70~+130 | +66~+126 |

→ 平身状态下墙距 100~200mm 就存在约 60mm 高的可落足带（femur 实测行程比官方假设
多 14.4° 上抬，近距离墙也可达）；身体俯仰 30°
（`robot.py` 的 `body_rpy` 已支持）后落足带更宽、更高、更近。过渡序列每一步
都有解，但序列编排复杂，仍按路线图放在 P5。

**结论与设计决策：**
1. 不需要给腿加第 4 个踝关节舵机——省下 6 个舵机的重量和复杂度。
2. 可行性的关键参数是波纹吸盘的实际角度容差。**P1 台架实测结果：斜角容差达到了 15°！** 这意味着我们可以满额释放机器狗的机动能力，采用 **40mm 的大步幅**进行爬墙，无需收紧到 25mm。
3. 以上用电气行程计算，未计结构干涉（femur 撞机身、tibia 撞 femur）——P2 单腿
   试验时实测各关节的真实机械行程并回填本分析。
